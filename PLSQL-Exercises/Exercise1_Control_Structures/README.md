# Exercise 1: Control Structures

This folder contains PL/SQL exercises demonstrating control structures like IF-THEN-ELSE, CASE, and LOOPS.

## Scenarios
| Scenario 1 | Apply 1% discount to customers above 60 years old | `Exercise1_Scenario1_Discount.sql` |
| Scenario 2 | Promote VIP customers with balance > $10,000 | `Exercise1_Scenario2_VIP.sql` |
| Scenario 3 | Send reminders for loans due in 30 days | `Exercise1_Scenario3_Reminders.sql` |

## Key Concepts Covered

- `IF-THEN-ELSE` statements
- `FOR` loops with cursors
- `CASE` statements
- Exception handling
- Database updates with `COMMIT` and `ROLLBACK`
